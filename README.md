# Tic-Tac-Toe game

A classic, fully playable Tic-Tac-Toe game built with Python and Pygame. 

## Features
* **Interactive UI:** Playable game board with bold graphics.
* **Player Choice:** Choose to start as Player X or Player O.
* **Automatic Saving:** Wins and draws are automatically saved to a `scores.json` file so you never lose your progress.
* **Leaderboard:** View an automatically sorted list of all-time scores directly in the game.
* **Quick Controls:** Press `R` to restart, `M` for the Menu, and `L` for the Leaderboard at any time.

1. **Activate your virtual environment:**

   * Mac/Linux: `source .venv/bin/activate`

2. **Install requirements:**

3.**Start the game:**
